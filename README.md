# Damdi

## Getting Started

1.  Clone the repo

```bash
$ git clone git@github.com:021NIS/Damdi.git
$ cd Damdi
$ rm -rf .git
$ git init
```

2.  Install dependencies from [npm](https://www.npmjs.com)

```bash
$ npm install
$ react-native link
$ react-native run-ios (or run-android)
```
